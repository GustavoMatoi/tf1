Desenvolvimento de um Sistema Multiplataforma Baseado em Reconhecimento de Gestos para Transferência de Arquivos Utilizando Tecnologias Web e Sistemas Distribuídos

Baseado no Huawei AI Air Transfer

Descrição do projeto:

O presente projeto propõe o desenvolvimento de um sistema multiplataforma para transferência de arquivos via reconhecimento de gestos, inspirado em soluções como o AI Air Transfer da Huawei. A proposta central é permitir que usuários transfiram arquivos entre dispositivos (web, desktop e mobile) de forma intuitiva e sem contato físico direto, apenas com movimentos captados por câmeras.

O sistema será desenvolvido utilizando Next.js, Electron e React Native, com a linguagem TypeScript como base, visando garantir alta performance, segurança e integração entre plataformas. Serão aplicados conceitos de sistemas distribuídos para gerenciar as conexões e trocas entre os dispositivos, bem como técnicas de visão computacional, com apoio de bibliotecas como MediaPipe ou TensorFlow.js, para reconhecer gestos pré-definidos de forma precisa.

Além de inovar na forma de interação com arquivos, o projeto também considera aspectos de acessibilidade, oferecendo uma alternativa tecnológica que pode beneficiar pessoas com limitações motoras ou em contextos onde o uso direto de periféricos é limitado.

A iniciativa busca demonstrar como tecnologias modernas de front-end e inteligência artificial podem ser combinadas para criar experiências mais naturais, eficientes e inclusivas, explorando o potencial de interfaces não convencionais em ambientes distribuídos.
